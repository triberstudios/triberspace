Layout Persistence Test

This test checks if the layout persistence functionality is working correctly.

Test Instructions:

  1. Open the Triber Editor (localhost:3001)
  2. Resize the sidebar width by dragging the horizontal resizer
  3. Resize the viewport height by dragging the vertical resizer
  4. Refresh the page
  5. Check if the layout maintains the same dimensions

Manual Test Results:

□ Sidebar width persists after refresh

□ Viewport height persists after refresh

□ Scrolling still works after layout changes

□ Sketchfab menu option is present
